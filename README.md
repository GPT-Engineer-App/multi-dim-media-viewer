# multi-dim-media-viewer

A component for viewing media that contains many (varying) dimensions or aspects that should need to be presented simultaneously - quick navigation between overview and aspect details as well as fine-grained step navigation between complex levels of detail.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/multi-dim-media-viewer.git

```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
